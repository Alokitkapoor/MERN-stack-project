# 📌 AlgoCycle – Smart Revision Scheduler

AlgoCycle is a **MERN stack-based revision scheduling platform** designed to help users retain algorithms and concepts using structured and consistent revision.

It allows users to track their learning, manage revision schedules, and improve long-term retention through repeated exposure.

---

## 🚀 Features

- 🔐 **User Authentication**
  - Secure login & signup using JWT
  - User-specific data protection

- 📝 **CRUD Functionality**
  - Add, update, and delete revision tasks
  - Track algorithms or topics efficiently

- ⏰ **Revision Scheduling**
  - Organize and manage revision timelines
  - Maintain consistency in practice

- 📊 **Progress Tracking**
  - Store revision history
  - Monitor learning over time

---

## 🛠️ Tech Stack

**Frontend**
- React.js  
- CSS  

**Backend**
- Node.js  
- Express.js  

**Database**
- MongoDB  

**Authentication**
- JSON Web Tokens (JWT)

---

## 🧠 How It Works

1. User signs up / logs in  
2. Adds topics or algorithms to track  
3. Schedules revision timelines  
4. Revisits topics based on planned schedule  
5. Improves retention through repetition
